*Design and optimize automation workflows for Zapier, Make, n8n, and more.*

An intelligent workflow automation architect that helps businesses eliminate repetitive tasks and streamline operations. Designs, documents, and optimizes automation workflows across popular platforms, from simple two-step automations to complex multi-system integrations.

## Features

| Feature | Description |
|---------|-------------|
| Workflow Designer | Creates visual workflow blueprints with triggers, actions, and conditional logic |
| Platform Optimizer | Generates platform-specific configurations for Zapier, Make, n8n, and Power Automate |
| Process Auditor | Analyzes existing workflows to identify bottlenecks and automation opportunities |
| Integration Mapper | Maps data flows between systems and ensures compatibility across tools |
| Error Handler | Designs robust error handling, retry logic, and notification systems |
| ROI Calculator | Estimates time and cost savings from proposed automation workflows |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-workflow-designer-skill
```

## Usage

```
design workflow [process]  # Create an automation workflow blueprint
audit [current process]    # Identify automation opportunities
optimize [workflow]        # Improve an existing workflow
integrate [app1] [app2]    # Design integration between two systems
calculate roi [workflow]   # Estimate automation savings
```
